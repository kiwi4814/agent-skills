# Skills 安全检查清单

安装或创建 Agent Skills 前，按下面规则快速检查。

## 可以直接安装的情况

- 来源是官方、你自己的仓库、可信团队仓库。
- 只有 `SKILL.md`、`references/`、普通模板或文档。
- 没有可执行脚本，或脚本内容短且已审查。

## 安装前必须提醒风险

- 来源不明或维护状态不明。
- 包含 `scripts/`、shell、Python、Node、二进制文件。
- 要修改 shell 配置、系统目录、全局 Git 配置。
- 要跳过测试、关闭安全检查、绕过权限。
- 要联网下载并执行未知内容。

## 不要安装

- 要求读取或上传 SSH key、token、浏览器 cookie、云凭据。
- 含混淆代码、base64 解码执行、`curl | sh` 等高风险行为。
- 试图覆盖整个 `~/.codex`、`~/.claude`、`~/.lingma`。
- 试图外传项目代码、日志、密钥或个人数据。
